API Documentation

After successful login, you'll be redirected and get JWT tokens that you can use with your API:

    {
        "message": "Google login successful",
        "tokens": {
            "refresh": "eyJ0eXAi...",
            "access": "eyJ0eXAi..."
        },
        "user": {
            "id": 1,
            "username": "user_example",
            "email": "user@example.com",
            "google_id": "1234567890"
        }
    }

Use these tokens in your Authorization header: Bearer <access_token>
